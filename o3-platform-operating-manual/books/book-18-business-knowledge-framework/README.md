# Book 18: Business Knowledge Framework

## Purpose

This book defines the business knowledge framework that powers O³'s domain-specific AI, including HR business rules, decision logic, action playbooks, and the Decision Knowledge Base (ODKB).

## Chapters

1. Business Knowledge Framework Overview
2. HR Domain Knowledge Base
3. Decision Rules and Logic
4. Action Playbook Library
5. Industry-Specific Knowledge
6. Thai SME Context Knowledge
7. KPI Interpretation Rules
8. Risk Assessment Framework
9. Recommendation Logic
10. Knowledge Base Maintenance

## Key Takeaways

- ODKB (O³ Decision Knowledge Base) stores HR business rules and decision logic
- Action playbooks provide templated responses for common scenarios
- Industry-specific knowledge enhances AI recommendations
- Thai SME context is a key differentiator for AI quality
- Knowledge base is maintained and versioned separately from code
- AI uses knowledge base to generate context-aware recommendations
