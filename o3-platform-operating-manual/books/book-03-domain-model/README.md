# Book 03: Domain Model

## Purpose

This book defines the business objects (domains) that make up the O³ platform before any database or API design. The domain model describes the world of O³ in business language.

## Key Takeaways

- Company is the root domain owning data, users, subscription, and product usage
- Employee is the core workforce object connecting to most insights
- Insight is the interpreted layer above raw data
- AI Advisor reads context and produces recommendations, does not own data
- Benchmark uses anonymized aggregated data only
