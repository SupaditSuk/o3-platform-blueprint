# Book 13: Dashboard Engine

## Purpose

This book defines the dashboard engine architecture, including widget patterns, KPI rendering, insight integration, AI explanation embedding, and action plan display.

## Chapters

1. Dashboard Philosophy (Insight, not BI)
2. Dashboard Page Architecture
3. Widget Pattern Standard
4. KPI Rendering Engine
5. Risk Level Visualization
6. AI Explanation Integration
7. Action Plan Display
8. Dashboard Configuration
9. Dashboard API
10. Dashboard Performance Optimization
11. Export and Report Generation

## Key Takeaways

- Dashboard is decision intelligence, not BI reporting
- Every widget includes: KPI name, value, trend, risk level, business meaning, AI explain button, suggested action, data source
- MVP dashboard pages: Executive Home, Workforce Snapshot, Turnover & Retention, Productivity, Talent & Performance, Learning, Sentiment
- Guided dashboards in MVP, not fully customizable
- AI summary embedded in dashboard, not separate
