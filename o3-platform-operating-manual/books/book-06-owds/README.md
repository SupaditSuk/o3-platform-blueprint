# Book 06: OWDS (O³ Workforce Data Standard)

## Purpose

This book defines the O³ Workforce Data Standard — the common data language for all data ingestion, dashboard, AI, survey, and benchmark operations.

## Chapters

1. OWDS Overview and Rationale
2. One Simple Company Data Template
3. Employee Master Schema
4. Exit Record Schema
5. Performance Schema
6. Training Schema
7. Business Output Schema
8. Employee Survey Schema
9. Data Quality Rules
10. Regrettable Loss Logic
11. Cost of Attrition Calculation
12. OWDS Versioning and Extension

## Key Takeaways

- OWDS standardizes field names across all customer data sources
- One Simple Template has 7 sheets covering core workforce data
- Regrettable loss detected through performance, potential, key talent, critical position indicators
- Data quality rules check required fields, duplicates, formats, and consistency
- OWDS is versioned and extended through ADR process
