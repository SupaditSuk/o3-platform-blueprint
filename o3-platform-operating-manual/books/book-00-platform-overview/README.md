# Book 00: Platform Overview

## Purpose

This book provides a high-level overview of the O³ ZONE platform, its products, architecture layers, and MVP scope. It serves as the entry point for anyone new to the platform.

## Chapters

1. Platform Concept and Products
2. Platform Layers
3. MVP Scope and Flow
4. Platform Positioning
5. Product Relationship Model (Assess → Advise → Accelerate)

## Key Takeaways

- O³ ZONE is a workforce intelligence platform, not a dashboard, HRIS, or chatbot
- Four main products: Dashboard, Workforce AI Studio, Survey Studio, Academy
- Five platform layers: Experience, Product, Intelligence, Data, Platform
- MVP starts with a simple data-to-insight journey
- All products share common foundations (OWDS, AI Gateway, Insight Engine, etc.)
