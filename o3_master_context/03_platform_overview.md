# 03 Platform Overview

## Platform Concept

O³ ZONE should be understood as a platform with multiple product surfaces but one shared foundation.

The four main products are:

1. O³ Dashboard
2. O³ Workforce AI Studio
3. O³ Survey Studio
4. O³ Academy

Future product:

5. O³ Benchmark Center

All products should share:

- Company profile
- User account
- Subscription and permission system
- AI Gateway
- OWDS data model
- Insight engine
- CRM and usage tracking
- Design system
- Admin configuration

## Recommended MVP Flow

The MVP should not start with a complex dashboard. It should start with a simple data-to-insight journey:

1. User creates account.
2. User enters company profile.
3. User downloads or uploads One Simple Company Data Template.
4. System validates data quality.
5. System imports data into OWDS.
6. System generates Workforce Snapshot.
7. System generates Turnover and Regrettable Loss insights.
8. AI summarizes key risks and actions.
9. User can upgrade for deeper analysis or AI tools.

## Platform Positioning Shift

Do not position the product as:

> A dashboard platform.

Position it as:

> A workforce intelligence platform that turns company data into insight and action.

## Core MVP Products

MVP should focus on:

1. Company Profile
2. One Simple Company Data Template
3. Upload and Validation
4. Workforce Dashboard MVP
5. Turnover and Regrettable Loss
6. AI Summary and Action Plan
7. Basic AI Tools
8. Basic Academy
9. Subscription and usage tracking

Do not build in MVP:

- Full Survey Studio
- Full Benchmark Center
- Enterprise HRIS integration
- Advanced marketplace
- Deep customization engine
- Complex BI builder
